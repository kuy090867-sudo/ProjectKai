# Project Kai — 상세 로드맵 v2.1
## 스토리 연동 (돈키호테 모티프)

---

## 평가 기준 (Dead Cells/Hollow Knight/Hades 기준)

| 점수 | 기준 |
|------|------|
| 1점 | 움직이는 사각형 |
| 2점 | 스프라이트+기본 물리 |
| 3점 | 전투 파이프라인 작동 (적 죽일 수 있음) ← **현재** |
| 4점 | 게임필 기본 (애니메이션, 이펙트, 타일맵, 피격 리액션) |
| 5점 | 1장 플레이 가능 (스토리 도입 + 3스테이지 + 보스) |
| 6점 | 거점 + UI 완성 + 사운드 + 폴리싱 |
| 7점 | 2장 완성 (기사단장 등장 + 패배 연출) |
| 8점 | 3장 완성 + 엔딩 + 밸런싱 |
| 9점 | 전체 폴리싱 + 스토리 연출 + 유저 테스트 |
| 10점 | 출시 가능 |

---

## Phase 1: 기반 — "검을 들기 전에" (3점 → 4점)
**스토리 없음. 순수 게임필 기반 작업.**

### 1-1. 피격 리액션 (가장 중요)
- [ ] 적 피격 시 흰색 플래시 (0.1초)
- [ ] 적 피격 시 스프라이트 뒤로 밀림 (넉백 시각화)
- [ ] 적 피격 시 히트스톱 정밀 튜닝 (일반 30ms, 강타 80ms)
- [ ] 적 사망 시 슬로우모션 (0.3초간 0.3배속)
- [ ] 적 사망 시 파티클 (파편 흩어짐)

### 1-2. 공격 비주얼
- [ ] 검 슬래시 이펙트 (호 형태 스프라이트)
- [ ] 공격 시 카메라쉐이크 (강도별)
- [ ] 콤보 3타 시 확대 이펙트
- [ ] 대시 잔상 효과

### 1-3. 캐릭터 애니메이션
- [ ] Player: Idle(4f), Run(4f), Jump(2f), Fall(2f), Attack1~3(각 3f), Dash(2f), Hit(2f), Death(4f)
- [ ] Goblin: Idle(4f), Run(4f), Attack(3f), Hit(1f), Death(3f)
- [ ] 애니메이션 상태 전환 연동

### 1-4. 타일맵 레벨
- [ ] DungeonTilesetII로 1-1 스테이지 타일맵 구성
- [ ] 배경 레이어 2개 (패럴랙스)
- [ ] 플랫폼/벽/천장 타일 배치

### 1-5. 사운드 교체
- [ ] 무료 SFX 다운로드 (OpenGameArt RPG Sound Pack)
- [ ] 합성음 → 실제 효과음 교체 (검/타격/점프/대시/피격/사망)

## Phase 2: 제1장 "풍차를 향해" (4점 → 5점)
**스토리 시작. 카이의 첫 의뢰.**

### 2-1. 스토리 시스템
- [ ] 대화 시스템 구현 (TextMeshPro 기반)
- [ ] 대화 UI (캐릭터 초상화 + 이름 + 대사 박스)
- [ ] 스테이지 시작 전 대화 트리거
- [ ] 1-1 도입 대사 구현:
  > 리나: "간단한 일이야. 폐허 던전에 고블린이 늘었대."
  > 카이: "슬럼 아이들이 위험하다고?"
  > 카이: "됐어. 가자."

### 2-2. 1-1 스테이지: 슬럼 외곽
- [ ] 타일맵으로 폐허 슬럼 구성 (무너진 건물, 네온+마법 문양)
- [ ] 고블린 3마리 배치 (기획서 기준)
- [ ] 튜토리얼 구간 (이동/점프/공격 안내 텍스트)
- [ ] 스테이지 클리어 조건: 모든 적 처치
- [ ] 클리어 시 다음 스테이지 전환

### 2-3. 1-2 스테이지: 슬럼 지하
- [ ] 지하 던전 타일맵 (횃불, 어두운 톤)
- [ ] 에테르 수정 고블린 (강화형, HP 높음)
- [ ] 플랫폼 점프 구간
- [ ] 중간 대사:
  > 카이: "이 수정... 군용이잖아."
  > 리나: "누가 일부러 고블린을 강화시킨 거야."

### 2-4. 1-3 보스: 수정왕 고블린
- [ ] 보스 전용 AI (3페이즈: 돌진/점프/분노)
- [ ] 보스 체력바 (화면 하단 고정)
- [ ] 보스 진입 연출: 이름 표시 "수정왕 고블린 — 그림자의 실험체"
- [ ] 보스 방 문구: **"풍차인가, 거인인가"**
- [ ] 보스 BGM
- [ ] 처치 후 대사:
  > 카이: "그림자 기사단... 슬럼을 실험장으로 쓴 거야."
  > 리나: "또 시작이야, 돈키호테 선생님."
  > 카이: "풍차가 아니면 좋겠지만."

## Phase 3: 거점 + UI (5점 → 6점)
**카이의 거점. Hades식 캐릭터 대화.**

### 3-1. 거점 씬 (Hub)
- [ ] 카이의 사무실/은신처 씬
- [ ] 리나 NPC (대화)
- [ ] 무기 강화 NPC (상점)
- [ ] 스테이지 선택 (1장/2장)
- [ ] 거점 대화 시스템 (매번 다른 대화):
  > 리나: "넌 왜 맨날 손해 보는 일만 하는 거야?"
  > 카이: "손해인지는 나중에 알겠지."

### 3-2. UI 완성
- [ ] 메인 메뉴 (시작/설정/종료)
- [ ] HUD: HP바(좌상단), 무기 아이콘, 스킬 쿨타임
- [ ] 일시정지 메뉴
- [ ] 사망 화면: **"라만차의 기사는 쓰러져도 다시 일어난다."** [재시도] [거점으로]
- [ ] 스테이지 클리어 결과 화면

### 3-3. 사운드 완성
- [ ] 스테이지별 BGM
- [ ] 보스 BGM
- [ ] 거점 BGM (평화로운 톤)
- [ ] 환경음 (던전 물소리, 바람)
- [ ] 발소리 (바닥 재질별)

### 3-4. 폴리싱
- [ ] 무적 프레임 (피격 후 0.5초)
- [ ] 스프라이트 깜빡임 (무적 중)
- [ ] 적 공격 텔레그래프 (공격 전 0.3초 붉은 빛)
- [ ] 착지 먼지 파티클
- [ ] 대시 먼지 파티클

## Phase 4: 제2장 "거울의 기사" (6점 → 7점)
**기사단장 등장. 카이의 첫 패배.**

### 4-1. 새 적 종류
- [ ] 스켈레톤 궁수 (원거리, 화살 발사, 후퇴 AI)
- [ ] 오크 워리어 (탱크, 느리지만 강함, 방패 차기)
- [ ] 에테르 마법사 (원거리, 마법 투사체, 순간이동)
- [ ] 각 적 고유 스프라이트 + 애니메이션

### 4-2. 2-1 스테이지: 기사단 외곽 기지
- [ ] 산업 던전 타일셋 (파이프, 마법 장치)
- [ ] 스켈레톤 + 오크 혼합 배치
- [ ] 함정 (바닥 가시, 회전 톱날)

### 4-3. 2-2 스테이지: 지하 연구소
- [ ] 연구소 타일셋 (실험 장비, 수정 탱크)
- [ ] 리나 부모의 연구 기록 (환경 스토리텔링)
- [ ] 에테르 마법사 첫 등장

### 4-4. 2-3 보스: 기사단장 (첫 만남)
- [ ] **이길 수 없는 보스전** (HP가 일정 이하로 안 내려감)
- [ ] 3페이즈 후 카이가 쓰러지는 컷씬
- [ ] 보스 방 문구: **"거울 앞에 선 기사"**
- [ ] 대사:
  > 기사단장: "기사도로 세상을 바꿀 수 있었나? 권력만이 바꿀 수 있다."
  > 카이: "풍차가 아니었어. 진짜 거인이었다."
  > 리나: "그래도 또 돌진할 거잖아."
  > 카이: "당연하지."
- [ ] 패배 후 거점으로 강제 이동 (스토리 진행)

## Phase 5: 제3장 "라만차의 기사" (7점 → 8점)
**카이의 재기. 최종 결전.**

### 5-1. 성장 시스템
- [ ] 경험치/레벨업
- [ ] 스탯 포인트 (STR/DEX/INT)
- [ ] 무기 강화 (검/총 업그레이드)
- [ ] 포션/소모품

### 5-2. 3-1 스테이지: 기사단 본거지 입구
- [ ] 거대 성 타일셋 (성벽, 깃발, 마법 방벽)
- [ ] 이전 적 강화 버전 등장
- [ ] 슬럼 사람들의 도움 연출 (문 열어줌, 아이템 제공)

### 5-3. 3-2 스테이지: 기사단 내부
- [ ] 호화로운 실내 + 어두운 지하 대비
- [ ] 기사단 정예병 (신규 적)
- [ ] 환경 스토리텔링 (기사단의 진짜 목적)

### 5-4. 3-3 최종 보스: 기사단장 (최종전)
- [ ] 이번에는 이길 수 있는 보스전
- [ ] 4페이즈 (이전 3 + 분노 모드)
- [ ] 보스 방 문구: **"라만차의 기사, 다시 일어서다"**
- [ ] 대사:
  > 기사단장: "용병 하나가 뭘 할 수 있지?"
  > 카이: "하나가 바꿀 수 없다는 거 알아. 그래도 시작은 하나부터잖아."
- [ ] 처치 후 엔딩 컷씬

### 5-5. 엔딩
- [ ] 대균열 장치 파괴
- [ ] 세상이 크게 바뀌진 않음 (현실적 엔딩)
- [ ] 마지막 대사:
  > 리나: "결국 뭐가 바뀐 거야?"
  > 카이: (슬럼 아이가 손을 흔드는 걸 보며) "저것만 바뀌었으면 됐어."
  > 리나: "...바보."
  > 카이: "그래, 라만차의 기사는 원래 바보야."
- [ ] 엔딩 크레딧 + 후속 암시
