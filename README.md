# PROJECT KAI — 라만차의 기사

> "세상이 미쳤다고 해도, 옳다고 믿는 것을 위해 싸우는 자가 기사다."

## 🎮 게임 소개
**2D 횡스크롤 액션 RPG** | Unity 6 | C#

현대와 중세 마법이 공존하는 세계 **신서울(Neo-Seoul)**을 배경으로,
기사도를 믿는 용병 **카이(Kai)**가 거대한 세력에 맞서 싸우는 이야기.

**돈키호테** 모티프를 현대 판타지에 녹여낸 스토리 드리븐 액션 게임.

## ⚔️ 핵심 특징

### 전투
- **DMC 스타일 콤보 시스템** — 3단 검 콤보 + 마법 권총
- **무기 전환** (Tab) — 검 ↔ 총 즉시 전환
- **에어 콤보** — 공중에서도 공격 가능
- **대시 무적** — 회피와 전투의 융합
- **콤보 카운터** — C/B/A/S/SS/SSS 랭크

### 적
- **고블린** — 근접 졸개
- **스켈레톤 궁수** — 원거리 후퇴형
- **오크 워리어** — 탱크 (방패 차기)
- **에테르 마법사** — 순간이동 + 투사체
- **수정왕 고블린** — 1장 보스 (3페이즈)
- **기사단장** — 2장 보스 (이길 수 없는 전투) / 3장 최종 보스

### 스토리 (돈키호테 모티프)
- **1장: 풍차를 향해** — 슬럼 구출, 그림자 기사단 발견
- **2장: 거울의 기사** — 기사단장과의 첫 만남, 패배
- **3장: 라만차의 기사** — 재기와 최종 결전

### 시스템
- 경험치 / 레벨업 / 스탯 분배 (STR/DEX/INT)
- 무기 강화 (검/총 업그레이드)
- 챕터 해금 진행
- HP 포션 / 아이템 드롭
- 상태이상 (독/화상/빙결/스턴)
- 난이도 자동 스케일링
- 세이브/로드

### UI
- 메인 메뉴, 거점 (Hades식 대화), 일시정지
- 체력바 (Player + Enemy + Boss)
- 콤보 카운터 (DMC 스타일)
- 사망 화면: *"라만차의 기사는 쓰러져도 다시 일어난다."*
- 크레딧 스크롤
- 설정 (SFX/BGM 볼륨)
